# Jolen Game Assets

This folder contains assets for the Jolen (Filipino Marble) game.

## Suggested Assets:
- `player_marble.png` - Blue glass marble sprite for player
- `target_marble_red.png` - Red glass marble sprite for targets
- `target_marble_green.png` - Green glass marble sprite for targets
- `target_marble_yellow.png` - Yellow glass marble sprite for targets
- `game_background.png` - Dirt/ground texture background
- `aim_cursor.png` - Custom cursor for aiming
- `marble_hit_sound.mp3` - Sound effect when marbles collide
- `marble_roll_sound.mp3` - Sound effect when marble is rolling
- `success_sound.mp3` - Sound effect when level is completed

## Notes:
Currently using procedural drawing with canvas. Add actual marble sprites here to enhance the visual experience.
